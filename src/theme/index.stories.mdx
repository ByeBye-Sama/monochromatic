## palette

**Monochromatic UI** as the name says is based on one primary color, 
other colors in the palette are modifications of `primary`.

<Story name="palette" >{stories.Palette()}</Story>

<br/>

#### Use

To use just import `theme` and select one of the options of `palette` 
(`primary`, `tint1`, `tint2`, `tint3`, `shade1`, `shade2`, `shade3`, `white`,`black`,`primary00`,`primary25`,`primary50`,`primary75`).

With `inline style`:

```
import React from 'react'
import { theme } from 'monochromatic-ui'

const Component = () => {
  return(
    <div style={{ background: ${theme.palette.primary}, color: ${theme.palette.black} }} />
  )
}

export default Component
```

With `styled-components`:

```
import React from 'react'
import styled from 'styled-components'
import { theme } from 'monochromatic-ui'

const Component = styled.div`
  background: ${theme.palette.primary};
  color: ${theme.palette.black};
`

export default Component
```

#### Theming

To change the `primary` color of palette, you just need to assign a new one. 
You can make this on the `root` of your project. Remember that assigning a new color 
will change the color of all components, we do this because we think of **Monochromatic UI** 
as a single color based library and we prefer it to be used like this. 
Of course you can still overwrite the classes manually or maybe have one color implementation per page.

```jsx
import React from 'react'
import { theme } from 'monochromatic-ui'

theme.palette.primary = '#000000'

export default function App() {
  return (
    ...
  )
}
```

## textStyle

`textStyle` is an utily to apply font styles. Applies `font-family`, `font-size`, `font-weight` and `line-height`.

<Story name="textStyle" >{stories.TextStyle()}</Story>

#### Use

To use just import `theme` and select one of the options of `textStyle` 
(`h1`, `h2`, `h3`, `h4`, `h5`, `h5`, `h6`, `body1`,`body2`).

```
import React from 'react'
import styled from 'styled-components'
import { theme } from 'monochromatic-ui'

const Component = styled.p`
  ${theme.textStyle('h1')}
`

export default Component
```

#### Theming

Same as `pallete` if you want to modify `textStyle` you need to assign a 
new `textStyles` object on root. Even if you only want to modify a particular property, 
use the whole object or some properties could stop working correctly. 
If you change the `font-family` don't forget to add it to your project

```jsx
import React from 'react'
import { theme } from 'monochromatic-ui'

theme.textStyles: {

    fontWeight: 400,
    fontSize: '18px',
    lineHeight: '27px',
```

## spacing

Use the `spacing` helper to create consistent spacing between the elements of your UI.
**Monochromatic UI** uses a recommended 8px scaling factor by default.

<Story name="spacing" >{stories.Spacing()}</Story>

#### Use

To use just import `theme` and use a number.

```
import React from 'react'
import styled from 'styled-components'
import { theme } from 'monochromatic-ui'

const Component = styled.div`
  height: ${theme.spacing(10)};
  padding: ${theme.spacing(1, 3)};
`

export default Component
```

## boxShadow

Add or remove shadows to elements with box-shadow utilities.

<Story name="boxShadow" >{stories.BoxShadow()}</Story>

#### Use

To use just import `theme` and use integers from `0` to `10`, if it is `0` it will remove shadows.

```
import React from 'react'
import styled from 'styled-components'
import { theme } from 'monochromatic-ui'

const Component = styled.div`
  ${theme.boxShadow(2)}
`

export default Component
```

## breakpoint

Utility to define media queries for responsive design. 
`breakpoint` utility works with `max-width` to define media queries.

<Story name="breakpoint" >{stories.Breakpoint()}</Story>

#### Use

To use just import `theme` and choose one of the breakpoint options (`xs`, `sm`, `md`, `lg`, `xl`).

```
import React from 'react'
import styled from 'styled-components'
import { theme } from 'monochromatic-ui'

const Component = styled.div`
  background: ${theme.palette.primary};

  // example of media query for mobile
  ${theme.breakpoint.xs`
    background: ${theme.palette.black};
  `}
`

export default Component
```

#### Theming

If you want to change breakpoints you can make it assigning new ones on root, 
just remember to use the whole object. Also this change will modify how `Hidden` 
component is display.

```jsx
import React from 'react'
import { breakpoints } from 'monochromatic-ui'

breakpoints = {
  xs: 360,
  sm: 600,
  md: 960,
  lg: 1280,
  xl: 1920
}

export default function App() {
  return (
    ...
  )
}
```

# Styled components

**Monochromatic UI** is build with `Styled Components`, so for basic theming or custom components you can use it.

```
import React from 'react'
import styled from 'styled-components'
import { Button } from 'monochromatic-ui'

const CustomButton = styled(Button)`
  background: '#000';
  color: '#fff';
`

export default CustomButton
```
