# IconButton

<p style={{fontSize: '22px'}}>Icon buttons are commonly found in app bars and toolbars.</p>

## Normal

Icons are also appropriate for toggle buttons that allow a single choice to be selected or deselected, such as adding or removing a star to an item.

`IconButton` use `Icon` component so works with ligatures too, so search for icons on [Material Design](https://material.io/resources/icons).
